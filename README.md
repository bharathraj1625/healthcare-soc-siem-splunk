# Healthcare-SOC-SIEM-Splunk
SIEM implementation using Splunk &amp; BOTSv1 dataset. Covers log ingestion, threat detection (brute force, malware, lateral movement), event correlation, custom alerts, and SOC dashboard development. Built to replicate real-world SOC operations using healthcare and enterprise security scenarios.

Project Title: SIEM Analysis with Splunk using BOTSv1 Dataset
Description:
This project demonstrates the implementation of a Security Information and Event Management (SIEM) system using Splunk, built as part of an MSc Cybersecurity program at Mangalore University.
The analysis is performed on the BOTSv1 (Boss of the SOC v1) dataset — a real-world attack simulation dataset developed by Splunk — covering attack scenarios such as brute force attempts, malware execution, and lateral movement.
Key areas covered:

Log ingestion and centralized monitoring in Splunk
Log analysis to identify malicious patterns and anomalies
Event correlation to detect multi-stage attacks and map the kill chain
Custom alert configuration for real-time threat notifications (e.g., threshold-based brute force detection)
Interactive SOC dashboard development for single-pane-of-glass visibility into attack patterns, user behavior, and system activity

Skills demonstrated: Splunk SPL, threat detection, incident investigation, alert tuning, SOC operations, SIEM workflows
Dataset: BOTSv1 by Splunk
Tools Used: Splunk Enterprise, BOTSv1 Dataset
